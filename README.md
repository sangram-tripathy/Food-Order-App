# Food-Order-App

Developed a RESTful backend API for a food ordering application using Node.js and Express.js.

Implemented API endpoints to retrieve available meals and place customer orders.

Managed application data using file-based persistence (JSON files) for meals and order storage.

Handled HTTP requests and responses using Express middleware and JSON parsing.

Configured CORS headers to enable secure frontend-backend communication.

Ensured proper HTTP status codes and error handling for reliable API behavior.

Served static assets and followed modular backend development practices.
